# Suggested Commands
- Run app locally: `venv/Scripts/python.exe -m streamlit run app.py --server.port 8501`.
- Alternate simple run after environment activation: `streamlit run app.py`.
- Install dependencies: `venv/Scripts/python.exe -m pip install -r requirements.txt`.
- Windows launcher: `start.bat`; stop helper: `stop.bat`.
- Build release on Windows: `build_release.ps1` or `build_release.bat`.
- Search files/content: prefer `rg --files` and `rg pattern` from project root.
