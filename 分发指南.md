## 🚀 如何分发给朋友

### 步骤 1：压缩文件夹

**方法一：使用 Windows 右键菜单**
1. 在资源管理器中找到 `AlphaScanner_Release` 文件夹
2. 右键点击该文件夹
3. 选择 "发送到" → "压缩(zipped)文件夹"
4. 会生成 `AlphaScanner_Release.zip` 文件

**方法二：使用 PowerShell 命令**
```powershell
cd E:\code\py-rich\day-day-check
Compress-Archive -Path AlphaScanner_Release -DestinationPath AlphaScanner_Release.zip
```

### 步骤 2：发送给朋友

将生成的 `AlphaScanner_Release.zip` 文件通过以下方式发送给没有编程基础的朋友：
- 电子邮件附件
- 微信/QQ 文件传输
- U盘拷贝
- 网盘分享

### 步骤 3：告诉朋友如何使用

只需告诉朋友以下简单步骤：

1. **解压文件**：右键 ZIP 文件 → "全部解压缩"
2. **启动程序**：双击 `start.bat` 文件
3. **打开浏览器**：访问 http://localhost:8501
4. **开始使用**：享受应用功能！

## 📝 注意事项

### 文件大小
- 压缩包大小约：**80-100 MB**（取决于压缩率）
- 解压后大小约：**105 MB**

### 首次启动
- 第一次启动可能需要 **10-30 秒**
- 后续启动会更快

### 端口占用
- 如果提示端口被占用，先双击 `stop.bat` 再重新启动

### 杀毒软件
- 某些杀毒软件可能会误报（因为 PyInstaller 打包的程序特征）
- 如遇到此情况，需要将程序添加到白名单
- 这是正常现象，不是病毒

## 🆘 常见问题解答

### Q: 朋友收到后需要安装什么吗？
A: 不需要！直接解压就能用，所有依赖都已打包在内。

### Q: 可以在 Mac 或 Linux 上使用吗？
A: 不可以。这个打包版本只适用于 Windows 系统。如需跨平台，需要在对应系统上重新打包。

### Q: 如何更新程序？
A: 
1. 您重新打包新版本
2. 发送新的 ZIP 文件给朋友
3. 朋友删除旧文件夹，解压新文件即可

### Q: 可以修改默认端口吗？
A: 可以。编辑 `start.bat` 文件，将 `8501` 改为其他端口号即可。

### Q: 如何让程序开机自启动？
A: 将 `start.bat` 的快捷方式放到 Windows 启动文件夹：
   - 按 `Win + R`，输入 `shell:startup`，回车
   - 将快捷方式复制到此文件夹
