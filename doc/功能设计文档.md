## 一、工具定位与目标

**核心目标**：
1. **全市场扫描**：每日盘后或盘中，基于多因子模型，从全A股中筛选出Top 20只强势/低估个股。
2. **实时信号标注**：对筛选出的个股，在K线图上叠加**均线、布林带**，并自动标注**建仓（B点）** 和**风险（S点/预警）** 信号。

**适用场景**：每日复盘选股、盘中辅助盯盘（非自动化下单）。

四、功能模块详细设计
4.1 数据引擎模块（极简缓存版）
数据源：使用 Tushare 或 AKShare（免费Token即可）。

获取逻辑：

每日首次启动时，拉取全市场股票列表（剔除ST、退市、北交所）。

拉取所有股票近 120个交易日 的日线数据（OHLCV），存入本地 SQLite 或直接存为 CSV/Parquet 缓存。

策略：当日只拉取增量数据（最近1天），避免API额度浪费。

4.2 全市场选股扫描器（核心功能一）
输入：全市场股票池 + 最新财务/行情数据。

因子选取（精简为4个核心因子）：

估值因子（PB）：市净率低者加分（剔除负值）。
动量因子（RPS/20日涨幅）：20日涨幅排名靠前（强者恒强）。
波动因子（ATR）：剔除波动过小的僵尸股，保留活跃标的。
财务因子（ROE）：净资产收益率大于8%的加分。
打分公式：综合得分 = 0.3*动量分 + 0.3*估值分 + 0.2*波动分 + 0.2*ROE分（采用百分制排名）。

输出结果：每日输出排名前 20 只股票（入选“股票池”） ，并展示其综合得分。

4.3 实时K线图与择时信号模块（核心功能二）
触发方式：用户点击“股票池”中的任意一只股票，立即绘制该股的K线图。

图表功能（基于Matplotlib或Plotly）：

主图：绘制美国K线（蜡烛图） + 5日均线 + 20日均线。

副图：绘制布林带（Bollinger Bands）（中轨20日均线，上下轨±2倍标准差）。

信号标注逻辑：

建仓提醒（买入信号 B点）：

当 5日均线上穿20日均线（金叉） 且 收盘价突破布林带上轨。

界面表现：在K线对应位置画一个 绿色向上的箭头，并弹出文字“B：突破建仓”。

风险提示（卖出/预警信号 S点）：

当 5日均线下穿20日均线（死叉） 或 股价跌破布林带下轨。

界面表现：在K线对应位置画一个 红色向下的箭头，并弹出文字“S：风险离场”或“跌破下轨预警”。

4.4 可视化与交互界面
布局设计：

左侧面板（股票池列表）：展示排名前20的股票代码、名称、综合得分。

右侧主面板（K线图区）：展示当前选中个股的K线图、均线、布林带及标注信号。

预警机制：

当盘中（或每日数据更新后）扫描到符合条件的股票，Windows/Linux系统弹窗通知 或 打印高亮红色/绿色日志。

高级扩展：可发送邮件或飞书/微信机器人消息（此项作为Optional）。

def scan_universe(date):

全市场筛子（多因子选股）
