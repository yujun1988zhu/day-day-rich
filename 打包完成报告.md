##  技术细节

### PyInstaller 打包参数
```bash
pyinstaller --onefile \
    --name="AlphaScanner" \
    --add-data "data;data" \
    --add-data ".streamlit;.streamlit" \
    --hidden-import=pandas \
    --hidden-import=numpy \
    --hidden-import=mplfinance \
    --hidden-import=matplotlib \
    --collect-all streamlit \
    app.py
```

### 包含的依赖库
- streamlit >= 1.30.0
- akshare >= 1.10.0
- pandas >= 1.5.0

- mplfinance >= 0.12.10b0

## ⚠️ 重要提示

### 关于杀毒软件误报
PyInstaller 打包的程序可能会被某些杀毒软件误报为病毒，这是因为：
1. 打包后的 exe 文件特征与某些恶意软件相似
2. 缺少数字签名证书

**解决方案：**
- 将程序添加到杀毒软件白名单
- 或使用代码签名证书（需购买，成本较高）
- 向朋友说明这是正常现象，不是真正的病毒

### 跨平台限制
当前打包版本仅适用于 **Windows 系统**。如需在其他平台使用：
- **MacOS**：需要在 Mac 上重新打包
- **Linux**：需要在 Linux 上重新打包

### 更新机制
目前没有自动更新功能。如需更新：
1. 修改源代码
2. 重新运行打包脚本
3. 发送新的 ZIP 文件给用户
4. 用户替换旧文件即可
