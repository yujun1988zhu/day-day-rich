# 打包 AlphaScanner 为独立可执行程序

本指南说明如何将 AlphaScanner 打包成独立的 .exe 文件，方便分享给没有编程基础的朋友。

## 📦 方案选择

### 推荐方案：PyInstaller + 便携包
- ✅ 生成单个 .exe 文件（或文件夹）
- ✅ 无需安装 Python
- ✅ 双击即可运行
- ️ 体积较大（约 200-400 MB）

---

## 🔧 步骤一：准备环境

### 1. 安装 PyInstaller
```bash
cd E:\code\py-rich\day-day-check
venv\Scripts\pip install pyinstaller
```

### 2. 确认依赖已安装
```bash
venv\Scripts\pip install -r requirements.txt
```

---

## 🚀 步骤二：执行打包

### 方式 A：单文件模式（推荐分享）
```bash
pyinstaller --onefile ^
    --name="AlphaScanner" ^
    --icon=NONE ^
    --add-data "data;data" ^
    --add-data ".streamlit;.streamlit" ^
    --hidden-import=pandas ^
    --hidden-import=numpy ^
    --hidden-import=mplfinance ^
    --hidden-import=matplotlib ^
    --collect-all streamlit ^
    app.py
```

**生成的文件位置：** `dist/AlphaScanner.exe`

### 方式 B：文件夹模式（启动更快）
```bash
pyinstaller ^
    --name="AlphaScanner" ^
    --add-data "data;data" ^
    --add-data ".streamlit;.streamlit" ^
    --hidden-import=pandas ^
    --hidden-import=numpy ^
    --hidden-import=mplfinance ^
    --hidden-import=matplotlib ^
    --collect-all streamlit ^
    app.py
```

**生成的文件夹位置：** `dist/AlphaScanner/`

---

## 📁 步骤三：整理发布包

### 创建发布目录结构
```
AlphaScanner_Release/
├── AlphaScanner.exe          # 主程序
├── start.bat                 # 启动脚本
├── stop.bat                  # 停止脚本
├── README_使用说明.md        # 使用文档
└── data/                     # 数据缓存目录（可选）
    └── watchlist.csv         # 自选股列表
```

### 简化版 start.bat（适配打包后）
```batch
@echo off
cd /d "%~dp0"

echo ========================================
echo    AlphaScanner 正在启动...
echo ========================================
echo.

REM 直接运行打包后的 exe
start "" /MIN AlphaScanner.exe

echo [成功] AlphaScanner 已启动
echo.
echo 访问地址: http://localhost:8501
echo.
timeout /t 3 >nul
```

## ❓ 常见问题

### Q: 打包后体积太大怎么办？
A: 
- 使用虚拟环境，只安装必要依赖
- 排除不需要的模块：`--exclude-module=tkinter`
- 使用 UPX 压缩

### Q: 朋友电脑报毒怎么办？
A: 
- PyInstaller 打包的程序可能被误报
- 建议添加到杀毒软件白名单
- 或使用代码签名证书（需购买）

### Q: 数据文件丢失怎么办？
A: 
- 确保 `--add-data` 参数正确
- 检查打包后的 `data/` 目录是否存在

### Q: 启动太慢怎么办？
A: 
- 使用文件夹模式而非单文件模式
- 首次启动需要解压，后续会快很多

## 💡 替代方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **PyInstaller** | 完全独立，无需Python | 体积大 | ✅ 推荐给小白用户 |
| **便携 Python** | 体积小，灵活 | 需手动配置 | 技术爱好者 |
| **在线部署** | 零安装 | 需网络，隐私风险 | 团队协作 |
| **Docker** | 环境隔离 | 需安装Docker | 开发者 |

**结论**：对于没有编程基础的朋友，**PyInstaller 单文件模式**是最简单、最友好的方案！
